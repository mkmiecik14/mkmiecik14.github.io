## About

This repository contains the source code for my personal academic website, where I blog about data science, statistics, R programming, and academic musings. The site serves as a platform to share research insights, tutorials, and thoughts on open science practices.

## Features

- 📝 **Blog Posts**: In-depth articles on data science, statistics, and R programming
- 📊 **Interactive Visualizations**: R-generated plots and analyses embedded in posts
- 📱 **Responsive Design**: Mobile-friendly layout with light/dark theme support
- 🔍 **Full-text Search**: Search across all blog content
- 💬 **Comments**: Giscus-powered discussion system
- 📈 **Analytics**: Google Analytics integration
- 🎓 **Academic CV**: Complete academic curriculum vitae
- 💼 **Consulting Info**: Statistical consulting services information

## Tech Stack

- **Framework**: [Quarto](https://quarto.org/) - Next-generation scientific publishing system
- **Language**: [R](https://r-project.org/) - Statistical computing and graphics
- **Styling**: Bootstrap (Flatly/Darkly themes)
- **Deployment**: GitHub Pages
- **Comments**: [Giscus](https://giscus.app/)
- **Analytics**: Google Analytics
- **Icons**: Bootstrap Icons + Academicons
